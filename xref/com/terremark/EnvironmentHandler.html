81       * <p/>
82       * {@code ProcessorCount} refers to the number of processors in the virtual machine. Permitted values are: 1, 2, 4,
83       * or 8.
84       * <p/>
85       * {@code Status} of {@code VirtualMachine} refers to the status of the virtual machine. Permitted values are:
86       * <ul>
87       * <li>NotDeployed - Host has not been defined but not deployed, such as with a physical device.</li>
88       * <li>Deployed - Host is defined and deployed, available for service.</li>
89       * <li>Orphaned - An attempt was made to create the host but failed.</li>
90       * <li>TaskInProgress - The host has been defined and creation is in progress.</li>
91       * <li>CopyInProgress - The host is to be a copy of an existing host and the copy is in progress.</li>
92       * </ul>
93       * Note: A virtual machine with {@code Status} {@code Orphaned} will contain only {@code Status}.
94       *
95       * @param environmentId The environment for which all compute pools should be retrieved.
96       * @return Compute pools in the specified environment.
97       * @throws TerremarkException If an error occurs retrieving the compute pools.
98       */
99      ComputePools getComputePools(String environmentId) throws TerremarkException;

398      * If successful and the name is changed, even if the description or tags are also changed, a task is returned.
399      * <p/>

408      * <p/>

413      * <p/>
414      * Note: If the high availability router has tags and {@code Tags} is empty, all tags are removed. Similarly, if any
415      * {@code Tag} exists on the high availability router and is not present in the call, that tag is removed from the
416      * high availability router. If the high availability router has tags and {@code Tags} is not present in the call,
417      * no change is made to the tags.
418      *
419      * @param obj High availability router to edit.
420      * @return Task, if the edit call was successful and name was updated. {@code Null} otherwise.
421      * @throws TerremarkException If an error occurs editing the high availablity router.
422      */
